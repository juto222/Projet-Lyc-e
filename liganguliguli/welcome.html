📚 Documentation Technique

Architecture du système

Le gestionnaire utilise une architecture en couches pour garantir la sécurité :

Couche de sécurité

PBKDF2HMAC : Dérivation de clé avec 100 000 itérations

Fernet : Chiffrement symétrique AES-256

SHA-256 : Hash du mot de passe maître

Couche de stockage

passwords_data.json : Mots de passe chiffrés

master_password.json : Hash + sel

Aucune donnée non chiffrée

Couche interface

Flet : Framework UI moderne

Python 3.11+

Design responsive

Classe PasswordManager

class PasswordManager: def __init__(self): self.data_file = "passwords_data.json" self.master_file = "master_password.json" self.cipher = None # Chiffrement Fernet def setup_master_password(self, password): # Génère un sel aléatoire salt = os.urandom(16) # Dérive une clé de chiffrement key = self.derive_key(password, salt) self.cipher = Fernet(key) def encrypt_data(self, data): # Chiffre avec AES-256 return self.cipher.encrypt(data.encode()).decode() def generate_password(self, length=16): # Génère un mot de passe sécurisé return ''.join(secrets.choice(characters) for _ in range(length))

❓ Questions Fréquentes

Que se passe-t-il si j'oublie mon mot de passe maître ?

Malheureusement, il est impossible de récupérer vos données. C'est le prix d'une sécurité maximale. Notez-le dans un endroit sûr !

Mes mots de passe sont-ils vraiment sécurisés ?

Oui ! Nous utilisons le même niveau de chiffrement que les banques (AES-256) et les agences gouvernementales.

Puis-je utiliser l'application sur plusieurs ordinateurs ?

Oui, copiez les fichiers passwords_data.json et master_password.json sur votre autre ordinateur.

L'application nécessite-t-elle Internet ?

Non ! Tout fonctionne 100% en local. Vos données ne quittent jamais votre ordinateur.

Combien de mots de passe puis-je stocker ?

Illimité ! Vous pouvez stocker autant de mots de passe que vous le souhaitez.
